# ED-IT!
Image Editor featuring a range of functionalities, such as:
dynamic filters, perspective changes, image rotation and
easy-to-use Drag & Drop option for image upload.

## How to run this project
- install dependecies:
<code>npm i</code>

- Run the apllication:
<code>npm start</code>

## Technologies
- HTML
- CSS
- JavaScript
- React
